# FontSafe Website

A simple, bubbly landing page for FontSafe by Beaker Lab.

## Setup

1. Upload the contents of this folder to your web server
2. Update the download URL in `script.js` (line 8) to point to your actual download file
3. The page will automatically use the app icon and hero image from UploadThing

## Features

- Responsive design that works on all devices
- Bubbly, modern design matching the app's aesthetic
- Smooth animations and hover effects
- Download button for macOS
- Support Us button linking to Buy Me a Coffee
- Linktree button

## Customization

- Update colors in `styles.css` by modifying the CSS variables in `:root`
- Change fonts by updating the Google Fonts import in `index.html`
- Modify button styles in the `.btn` classes in `styles.css`
